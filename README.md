## 🗺️ The 5-Phase Path

```
[01] IT & Security Basics   →   [02] GRC Frameworks   →   [03] Risk Management
                                                                      ↓
                              [05] Job Hunt   ←   [04] Policies & Auditing
```

| Phase | Focus | Duration | Key Cert |
|-------|-------|----------|----------|
| 01 | Networking, OS, Security Basics | 4–6 wks | Google IT / ITF+ |
| 02 | NIST, ISO 27001, SOC2, HIPAA | 6–8 wks | ISC2 CC (Free!) |
| 03 | Risk Assessment, FAIR, TPRM | 4–6 wks | — |
| 04 | Policies, Controls, Auditing | 4–5 wks | CISA Prep |
| 05 | Portfolio + Job Applications | Ongoing | CISM / CISSP |
